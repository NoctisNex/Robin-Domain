## ğŸ“± Responsive Design

The website is fully responsive using Tailwind's mobile-first approach:

- **Mobile** (`sm:`): Optimized navigation and typography
- **Tablet** (`md:`, `lg:`): Multi-column layouts and expanded content
- **Desktop** (`xl:`): Full-width grid systems (RPG status panel)
- **Fluid Typography**: `clamp()` and Tailwind responsive classes
- **Touch-Friendly**: Large tap targets and swipe-friendly layouts
- **Flexible Images**: Responsive scaling with `object-cover`

## ğŸš€ Getting Started

### Prerequisites

- A modern web browser
- No additional dependencies required

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/NoctisNex/Robin-Domain.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Robin-Domain
   ```

3. Open `index.html` in your web browser or use a local server:

   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```

4. Visit `http://localhost:8000` in your browser
